# Hashing 
Hashing is a technique used to store and retrieve data in **O(1)** time using a hash table
- Hashing is implemented in java : 1. HashMap
                                   2. HashSet
## Use :
- Need fast Search
- Frequency counting required
- Duplicate detection
- Pair finding problems
- Lookup optimization

# Algorithm
Step 1 : Create HashMap or HashSet
Step 2 : Traverse the array
Step 3 : Store element or frequency
Step 4 : Check required condition
Step 5 : Return result

#Time Complecity : O(n)
# Space Complecity : O(n)
